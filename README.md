## 🍏 What is mango-waf?

mango-waf is a tool designed to protect your network from certain types of cyber attacks. Specifically, it focuses on stopping Layer 7 Distributed Denial of Service (DDoS) attacks. These attacks try to overload your website or service with too many requests, making it unavailable for real users.

This software includes a simple dashboard that shows you live data about your network’s status. You do not need to be a programmer to use it. The goal is to provide an effective shield against attacks while keeping the tool easy to run on Windows computers.

## 💻 System Requirements

Before you start, make sure your computer meets these needs:

- Windows 10 or later (64-bit preferred)
- At least 4 GB of RAM
